Terminal Brief Summary

First terminal command we are going to learn is ls. This command can be run with or without arguments. You may run this command and press Enter to see output, which varies by the arguments passed with "-" flag.

  • ls — lists all contents in the folder inline.
  • ls -al — list all contents in the folder, but for each file/folder in new line. Output also includes dot (ex. .git/.gitignore/.vimrc) and hidden (ex. .DS_Store) files with permissions levels, dates.
  • ls -sl — list all contents in the folder, but for each file/fo lder in new line. Output includes permission levels, file sizes and dates.

Second command is to create empty file/folder. Terminal commands to create and remove folder are mkdir and rmdir respectively. Examples:

  • mkdir ~/Downloads/Test — creates empty folder "Test" in "Downloads" folder.
  • rmdir ~/Downloads/Test — removes folder "Test" from "Downloads" folder.
.

Third group of commands is to manipulate files, their contents, locations, etc.

  • touch ~/Desktop/test.txt — creates empty file in Desktop folder.
  • echo "Something" >> ~/Desktop/text.txt — writes word "something" in the file test.txt.
